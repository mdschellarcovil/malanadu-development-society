# Malanadu Development Society Website

A responsive, multi-page website for Malanadu Development Society in Chellarcovil, Idukki, Kerala.

## Structure

- `index.html` — homepage
- `pages/about.html` — about page
- `pages/products.html` — products page
- `pages/gallery.html` — gallery page
- `pages/contact.html` — contact page
- `css/style.css` — shared stylesheet
- `js/main.js` — navigation and reveal animations
- `assets/` — hero and placeholder SVG illustrations
- `logo.jpeg` — site logo image

## Features

- Modern dairy and agriculture theme
- Green and white color palette
- Fully responsive design for mobile, tablet, and desktop
- Smooth reveal animations and mobile menu
- SEO-friendly metadata and structured content
